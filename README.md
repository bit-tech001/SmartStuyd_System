<h1>🚀 SmartStudy System</h1>

<p>
A modern full-stack Study Management System built using the MERN Stack. The platform helps students and administrators manage assignments, exams, study materials, submissions, reports, and performance tracking through a responsive dashboard interface.
</p>

<h2 align="center">⚙️ Installation Guide</h2>

<h3>1️⃣ Clone Repository</h3>

<pre>
git clone https://github.com/bit-tech001/SmartStuyd_System.git
cd SmartStuyd_System
</pre>

<h3>2️⃣ Install Dependencies</h3>

<h4>Frontend</h4>

<pre>
cd client
npm install
</pre>

<h4>Backend</h4>

<pre>
cd server
npm install
</pre>

<hr/>

<h2 align="center">🌍 Environment Variables</h2>

<p>Create a <code>.env</code> file inside the server folder.</p>

<pre>
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
</pre>

<hr/>

<h2 align="center">▶️ Run Application</h2>

<h3>Start Backend Server</h3>

<pre>
cd server
npm run dev
</pre>

<h3>Start Frontend</h3>

<pre>
cd client
npm run dev
</pre>
